# Описание проекта
Викторина с одним экраном
![[квиз план]]
## Интерфейс
- Главное окно
- Шкала прогресса
- Текст вопроса
- Варианты ответов

## Функционал
- Вывод вопроса и ответов на экран
- Смена вопроса при нажатии на кнопку ответа
- Счетчик верных ответов
- Изменение размера шкалы прогресса при нажатии на кнопку ответа
